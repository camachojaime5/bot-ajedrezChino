# [Ajedrez Chino](https://ajedrezchino.es/como-anotar-jugadas-de-ajedrez-chino/)

### Navegación por el repositorio ##

<p>En este repositorio podrás encontrar tanto el código fuente como la documentación asociada al proyecto. Pero, para poder obtener información útil tendrás que entender la estructura del propio repositorio. La estructura de ramas es la siguiente:
  <ul>
    <li><b>Main</b>: Rama principal donde se encuentran las releases oficiales.</li>
    <li><b>Development</b>: Rama de desarrollo donde se encuentra el código que se está desarrollando en el ciclo actual.</li>
    <li><b>Ramas de características</b>: Reciben nombres con un identificador asociado a la tarea donde se crearon y que incluso las relaciona con su objetivo. Son las ramas donde cada uno de nosotros ha desarrollado el código relacionado con cada una de las pequeñas funcionalidades que integradas conforman el proyecto.</li>
  </ul>
</p>
